# Christine Liang 作品集網站 - 設計風格規劃

## 設計方案一：深色科技極簡主義
**設計運動**: Cyberpunk Minimalism + Dark Mode Elegance
**核心原則**:
- 深色背景（#0F1419）搭配高對比白色文字，營造科技感與專業度
- 金色（#D4AF37）與藍色（#00D9FF）作為強調色，呼應作品集中的遊戲與金融專案
- 大面積留白與卡片式設計，強調內容層級感
- 極簡線性圖標與精細排版，避免繁雜裝飾

**色彩哲學**:
- 背景：深灰黑色系，降低眼睛疲勞，提升專業感
- 強調色：金色用於 CTA 按鈕與重點標題，藍色用於互動元素
- 文字：白色主文本，灰色次文本，形成清晰層級

**版面範式**:
- 導覽列：固定頂部，深色背景，金色 Logo 文字
- Hero Section：全寬背景圖片（科技感抽象圖），左側文字堆疊，右側 CTA 按鈕
- 作品卡片：深色背景卡片，圖片頂部，標題+技術標籤下方，懸停時微妙浮起效果
- 頁尾：深色背景，簡潔聯絡資訊

**簽名元素**:
- 細線分隔符（Divider）貫穿各區域
- 金色邊框強調重點卡片
- 微妙的漸層背景（深灰→更深灰）

**互動哲學**:
- 按鈕懸停：金色背景 → 白色文字，平滑過渡
- 卡片懸停：輕微上浮，陰影加深
- 分類篩選：選中按鈕金色背景，未選中灰色邊框

**動畫指南**:
- 頁面進入：淡入 + 輕微向上位移（200ms）
- 卡片進入：錯開延遲，從下方淡入（300-500ms）
- 按鈕互動：快速反饋（100ms），金色脈動效果
- 分類篩選：平滑過渡，無突兀跳動

**排版系統**:
- 標題字體：Playfair Display（優雅襯線字）- 用於大標題
- 副標題：Poppins Bold（現代無襯線）- 用於區域標題
- 正文：Inter Regular（高可讀性）- 用於描述文本
- 技術標籤：Roboto Mono（等寬字體）- 用於技術棧顯示

## 設計方案二：明亮極簡專業風格
**設計運動**: Swiss Design + Minimalist Brutalism
**核心原則**:
- 白色背景搭配深灰文字，營造清爽專業感
- 黑色與淺灰色作為主色調，配合少量彩色強調
- 精確的網格系統與對齐，強調秩序感
- 大膽的排版與充足的負空間

**色彩哲學**:
- 背景：純白色，營造高級感
- 強調色：深黑色用於標題，淺灰色用於次級內容
- 點綴色：單一彩色（如深藍或深紫）用於 CTA 按鈕

**版面範式**:
- 導覽列：白色背景，黑色文字，簡潔線性
- Hero Section：左側大標題 + 描述，右側單色插圖或照片
- 作品卡片：白色卡片，邊框分隔，極簡排版
- 頁尾：白色背景，中央對齐，簡潔聯絡方式

**簽名元素**:
- 粗黑色線條分隔符
- 大膽的黑色標題
- 充足的負空間

**互動哲學**:
- 按鈕懸停：背景色變化，無過度動畫
- 卡片懸停：邊框加粗或背景變淡
- 分類篩選：黑色背景 → 白色文字

**動畫指南**:
- 頁面進入：簡潔淡入，無複雜位移
- 卡片進入：輕微縮放（1 → 1.02）
- 按鈕互動：快速背景色變化（80ms）
- 分類篩選：平滑顏色過渡

**排版系統**:
- 標題字體：Montserrat Bold（幾何無襯線）
- 副標題：Montserrat SemiBold
- 正文：Lato Regular（高可讀性）
- 技術標籤：IBM Plex Mono

## 設計方案三：漸層玻璃態現代風格
**設計運動**: Glassmorphism + Gradient Dynamics
**核心原則**:
- 彩色漸層背景（紫→藍→青），營造動態現代感
- 半透明玻璃卡片（Glassmorphism），背景模糊效果
- 豐富的漸層與光影效果，強調深度感
- 圓潤的邊角與柔和的陰影

**色彩哲學**:
- 背景：紫色漸層 → 藍色 → 青色，營造流動感
- 卡片：半透明白色（rgba(255,255,255,0.1)），背景模糊
- 強調色：金色與白色，在玻璃卡片上形成高對比
- 文字：白色主文本，淡白色次文本

**版面範式**:
- 導覽列：半透明玻璃背景，白色文字
- Hero Section：全寬漸層背景，中央堆疊文字與 CTA
- 作品卡片：玻璃態卡片，圖片頂部，標題+標籤下方
- 頁尾：深色漸層背景，白色文字

**簽名元素**:
- 漸層色彩過渡
- 玻璃態模糊效果
- 柔和的光線高光

**互動哲學**:
- 按鈕懸停：漸層色變化，光線高光移動
- 卡片懸停：玻璃效果加強，背景模糊增加
- 分類篩選：漸層背景 → 白色邊框

**動畫指南**:
- 頁面進入：漸層背景動畫，卡片淡入
- 卡片進入：玻璃效果漸進顯現
- 按鈕互動：漸層色流動效果
- 分類篩選：平滑漸層過渡

**排版系統**:
- 標題字體：Outfit Bold（現代幾何）
- 副標題：Outfit SemiBold
- 正文：Quicksand Regular（圓潤友善）
- 技術標籤：JetBrains Mono

## 推薦方案
**選擇：方案一 - 深色科技極簡主義**

理由：
1. 完美契合您作品集的視覺風格（遊戲、金融、體育專案都採用深色主題）
2. 金色強調色與您作品中的設計語言一致
3. 科技感與專業度兼具，適合 UI/UX 設計師與前端工程師的身份
4. 深色背景降低眼睛疲勞，提升長時間瀏覽體驗
5. 卡片式設計與大面積留白，強調作品內容本身
